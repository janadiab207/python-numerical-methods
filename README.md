# Python Numerical Methods

Python implementations of numerical methods developed as part of
Mathematics & Data Science coursework.

## Included Methods

### Time Stepping for ODEs
- Explicit Euler Method
- Runge-Kutta 4th Order (RK4)

### Square Root Approximation
- Iterative Taylor series expansion
- Error-based stopping criterion

### Legendre Polynomials
- Recursive computation of Legendre polynomials
- Visualization using Matplotlib

## Tools & Libraries
- Python
- NumPy
- Matplotlib

## Notes
These implementations focus on numerical accuracy, clarity, and algorithmic design.
